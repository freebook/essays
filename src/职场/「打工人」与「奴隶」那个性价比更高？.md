# 「打工人」与「奴隶」那个性价比更高？

你是否思考过？现代「打工人」与「古代奴隶」对于资本家哪个性价比更高？

按我们历史教科书里的主流五阶段论，人类社会形态的顺序是这样的：
原始社会
奴隶社会
封建社会
资本主义社会
共产主义社会（第一阶段是社会主义社会）

为什么西方资本主义国家近代又回到了奴隶社会时代？做起了贩卖黑奴的生意。而中国却早早的结束了奴隶社会进入封建社会？
上周开学送女儿返校，这是我给女儿留的深度思考问题。（我每次送她去上学，都会给坐在副驾女儿留一个深度思考作业）
结果女儿经过反复深度思考，还是没有给出我满意的答案，我想她的历史老师也无法回答这个问题。
首先，西方并没有回到奴隶时代。西方资本主义国家在近代进行黑奴贸易，并不是真正回到了奴隶社会，而是在资本主义发展过程中，出现了极端残酷、不人道的剥削方式。
其次，虽然黑奴贸易在形式上和古代奴隶社会有些相似，但近代的黑奴更多是被当作廉价劳动力，在种植园和工商业中从事高强度劳动。他们并非传统意义上的努力，而是被视为会说话的劳动工具，被强制劳动、被随意买卖，完全没有人身自由和基本权利。
现代「打工人」与「古代奴隶」哪个性价比更高？

很多人存在一个误区，使用奴隶更便宜，我女儿就是这种想法。其实不然，员工比奴隶成本更低。
Image
黑奴并非自愿劳作，而是在暴力逼迫下被迫劳动，完全没有人身自由与个人意志。使用奴隶，需要陪一个监工（人工成本），他们不会主动干活，对主人不忠诚，不懂感恩，主人要负责他的吃喝拉撒，生病了，要给他们治疗，否则生病死了，那就是财产损失。
为了有更多的年轻奴隶，还要给他们购买一些女奴隶负责生娃，女奴生产期间又无法参与生产劳动。出生的婴儿奴隶有生长期，在没有成为成年之前，还需要女奴照顾，这期间要消耗1.5个人力，并且无法创造价值，所以奴力性价比极低。
奴隶是一次性消耗品，就像是汽车，每天都在贬值，越老越不值钱，后期维护保养更贵，最后报废还要自掏腰包。奴隶从出生到老死，奴隶主要一直供他们吃穿住，生病也要治疗，相比还是使用农民工更划算。
中国人做为天选子民相比奴隶，性价比更高，干活更主动自驱，只要他给口饭吃，对东家感恩，工作抗压能力强，吃饭比牲口少，成本更低。
为什么西方资本主义国家近代又回到了奴隶社会时代？

前面已经讲了，使用奴隶的成本远远高于员工，为什么西方会选择历史倒退这个脚本呢？
原始→奴隶→封建→资本主义→奴隶→资本主义
答案：人力资源短缺
人力资源短缺，是指在一定时期内，社会或企业对劳动力的需求数量，大于市场上能够提供、且愿意从事相关工作的劳动力供给，从而出现岗位空缺、人手不足的现象。
而西方面临是的不是“短缺”是“没有”，短缺可以通过提高薪酬来解决，当年殖民地的工资水平是高于本国的，因为工资高低取决于经济发展速度和用工需求。但是“没有人”在短期是无法解决的。
殖民地的居民都是移民者，需要大批移民前往殖民地工作有个前提，就是本国有足够的人口数量，同时在殖民地工作能获得较好的收入，问题是西方国家本国居民人口数量不足，无法为殖民地输出劳动力。
在这方面，我国其实早有历史先例。几乎每个朝代，我们都会组织向边疆地区移民（戍边屯田）或者地广人稀的城市移民，填补那里的劳动力，而这一切的前提是，我们拥有足够庞大的人口基数，才有富余的人口可以向外迁移。
靖康之难・宋室南渡，累计南迁约500 万人，持续一个半世纪。明初，洪洞大槐树移民（山西 → 冀鲁豫皖苏）、江西填湖广（江西 → 两湖），民约1100 万人。清初湖广填四川，累计入川约500 万人。闯关东，清末累计移民超1000 万人。
解放黑奴，表面上看是上帝的恩赐与文明的进步，但从经济角度来看，真正的原因是殖民地经济发展吸引了大量投资者，外来自由劳动力不断涌入，填补了劳动力市场的空缺。
当低成本、可自由流动的雇工逐渐取代高成本、低效率的奴隶劳动时，奴隶制度便失去了原有的经济价值。
因此，黑奴获得解放，本质上是因为他们在经济上不再具备剥削价值，才最终被放弃、获得自由。
